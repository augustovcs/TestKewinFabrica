# Conleite — Site Institucional

> Do campo para sua mesa. Um site moderno e interativo para a fábrica de laticínios Conleite, inspirado no estilo premium da Piracanjuba.

## 🚀 Tech Stack

- **[Astro](https://astro.build/)** — Framework de alto desempenho com SSG
- **[React](https://react.dev/)** — Componentes interativos
- **[Tailwind CSS v4](https://tailwindcss.com/)** — Design system utilitário
- **[Framer Motion](https://www.framer.com/motion/)** — Animações premium
- **[Lucide React](https://lucide.dev/)** — Ícones elegantes

## ✨ Features

- 🎨 Design premium com tipografia Outfit e paleta da marca Conleite
- 🌊 Transições fluidas com wave dividers entre seções
- 🪄 Header transparente que vira glass morphism no scroll
- ⚡ Hero full-screen com animações staggered e shimmer text
- 📊 Cards de estatísticas com efeito glassmorphism flutuante
- 🎠 Carrossel de produtos com animações direcionais
- 📱 Design 100% responsivo (mobile-first)
- 🔍 SEO otimizado com meta tags e Open Graph

## 🧞 Comandos

| Comando             | Ação                                    |
| :------------------ | :-------------------------------------- |
| `npm install`       | Instala dependências                    |
| `npm run dev`       | Inicia servidor dev em `localhost:4321` |
| `npm run build`     | Build de produção em `./dist/`          |
| `npm run preview`   | Preview do build local                  |
